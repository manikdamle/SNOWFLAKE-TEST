Frequently Asked Questions

What is Snowflake?

Snowflake is a system that allows people to access censored websites and applications. It disguises your internet activity to look like regular video or voice calls, helping you bypass internet censorship.

How does Snowflake work compared to other tools?

Snowflake and similar tools transform internet activity to make it harder for censors to detect. Some tools redirect traffic to look like it's coming from cloud providers like Microsoft Azure or AWS, while others scramble traffic to appear random. Blocking these tools would require censors to block large portions of the internet, making it a costly endeavor.

Do I need to install an application to use Snowflake?

No, unlike VPNs, Snowflake is embedded within existing apps like Tor Browser (Desktop/Android), Onion Browser (iOS), and Orbot (Android/iOS). You just need to activate Snowflake in the settings of these apps if they are censored in your country.

How can I help others bypass censorship?

You can volunteer to operate a Snowflake proxy. Snowflake proxies are run by volunteers, and you can help users bypass censorship by running one. The only requirement is that the internet in your country isn't heavily censored.

Is my browsing activity exposed when I run a Snowflake proxy?

No, as a proxy operator, your IP address will not be exposed as the browsing IP for users connected through your proxy. The users' IP addresses will match their Tor exit node, not yours.
